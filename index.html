创建 Swap 文件
发表于2025-12-24|更新于2025-12-24|Linux|内存管理•Debian•Swap文件| 条评论
好的～要给你的 Debian 创建一个 4GB 的 swap 文件，很简单哦～一步步来： # 1. 创建 4GB 的 swap 文件（用 fallocate 更快） sudo fallocate -l 4G /swapfile # 如果 fallocate 不可用，就用 dd（比较慢） # sudo dd if=/dev/zero of=/swapfile bs=1M count=4096 status=progress # 2. 设置权限（很重要！） sudo chmod 600 /swapfile # 3. 设置为 swap 格式 sudo mkswap /swapfile # 4. 立即启用 sudo swapon /swapfile # 5. 开机自动挂载（编辑 /etc/fstab） echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab # 6. 检查一下～ swapon --show free -h 搞定啦～现在你应该有 4GB swap 了～需要我帮你看看效果吗？♡
